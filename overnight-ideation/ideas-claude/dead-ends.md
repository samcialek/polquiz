# Dead ends — explored and rejected

**Scope:** Ideas a cycle developed and then a later cycle (via CRITIQUE) determined are not worth pursuing. Logging these here prevents future cycles from re-proposing them.

**Format per entry:**

```
## [timestamp — cycle N — outcome]
**Idea (originally proposed cycle M, file: <file>):** <one-paragraph description>
**Why rejected:** <specific reason — cite evidence / file / counter-argument>
**What to do instead (if anything):** <one line or "drop entirely">
```

**Rules:**
- Don't dead-end an idea on hand-wave reasons ("seems risky"). Reject only with specific cited evidence.
- If an idea is "narrow and refine, not reject," keep it in the original file as a refinement; don't move it here.
- If two cycles disagree, log the disagreement in `cross-links.md`, not here.
