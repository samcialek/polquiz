# Existing-question rewrites

**Scope:** Questions in `src/config/questions.representative.ts` whose prompt wording, option phrasing, or evidence map should change. NOT new questions — those go in `new-questions.md`. NOT pure schema audits — those go in `audit-findings.md`.

**Each entry must include:**
- Question ID
- Verbatim quote of the current prompt or option text
- Specific proposed wording
- Reason: what failure mode does this fix? (ambiguity / overload / sign confusion / cross-population conflation)
- Risk: what could the rewrite break?
