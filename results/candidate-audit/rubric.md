# Candidate Coding Rubric (PRISM 14-node)

**Authority:** This document is the canonical reference for coding *historical political candidates* (1789–2024) on PRISM nodes. It is **not** a rubric for archetypes (those have their own activation-salience structure) and **not** a rubric for respondents (those are inferred from the quiz, not assigned).

**Created:** 2026-04-26 as Phase 2 of the candidate audit. Supersedes the implicit/drift rubric that produced the Pattern A (anti-establishment ≠ anti-institutional) and Pattern B (rhetorical universalism ≠ practiced moral scope) miscodings.

**Independence test.** Before coding any node value, ask: *would a careful historian, without seeing my other 13 values for this candidate, recognize this value as defensible?* If the value only "fits" because it makes the overall vector clean, it's overfit. Resist the urge to lock candidates into archetype-shaped vectors — they are political *signals*, not perfect type-specimens.

**Scope of "signal."** Each value codes the political signal a candidate sent during the campaign of that election year — platform, rhetoric, persona, and known actions in office through that point — not their private beliefs and not their later evolution. A candidate who shifted across cycles can have different vectors for different years.

## ENDS Cluster

### MAT — Material Orientation
**Pole 1:** redistribution / egalitarian (left-economic; the state should narrow material outcomes)
**Pole 5:** free-market / laissez-faire (right-economic; outcomes should be set by markets/property)
**Direction only.** State *scope* (how much the state can do at all) lives in ONT_S, not here. A laissez-faire libertarian and a small-government Jeffersonian both score high MAT, but they differ on ONT_S.

| Score | Anchor examples |
|-------|-----------------|
| 1 | Debs 1912, Norman Thomas, Bernie 2016/2020, FDR 1936, H. Wallace 1948 |
| 2 | Truman 1948, Carter 1976, Mondale 1984, Obama 2008 |
| 3 | Clinton 1992, Biden 2020, Eisenhower 1956 |
| 4 | Reagan 1980, Bush 1988, Romney 2012, Jefferson 1800 (anti-Hamilton, pro-agrarian-property) |
| 5 | Goldwater 1964, Ron Paul, Cleveland 1888 (gold-standard hard-money) |

**Common miscoding traps:**
- **Anti-Bank / anti-Hamilton ≠ pro-redistribution.** Jacksonians wanted hard money and property protection, not to narrow material outcomes. → MAT 3-4, not 1.
- **"Common man" populism ≠ MAT-low** unless coupled with redistributive policy. Anti-elite ≠ pro-egalitarian-economics.
- **Anti-corporate rhetoric** without specific redistributive policy → MAT mid (3).
- **Tariffs / protectionism** is a MAT-mid signal, not strongly polarized.

---

### CD — Cultural Direction
**Pole 1:** progressive / liberalizing on social issues (reproductive rights, LGBTQ, gender, marriage, religious accommodation, secular public sphere)
**Pole 5:** traditional / restorationist (return to past social arrangements, religious public sphere, traditional family/gender)

| Score | Anchor examples |
|-------|-----------------|
| 1 | AOC, Buttigieg, Bernie 2020, modern progressive D platform |
| 2 | Obama 2012, Clinton 2016, mainstream liberal D |
| 3 | Clinton 1992 (DLC), Eisenhower, mid-20th-century centrists |
| 4 | Reagan 1980, Bush 2004 (gay marriage ban), traditional R mainstream |
| 5 | Falwell-style movement leaders, Robertson, MAGA-era restorationists |

**Common miscoding traps:**
- **CD ≠ CU.** CD is *direction over time* on a contested set of issues; CU is *breadth at one moment* across ways of life. A 1796 candidate cannot be CD=1 in modern terms because the issue set was different. Code relative to the period's contested social-direction questions.
- **Religious tolerance** is CD-progressive (1-2 territory) but rarely max-1 unless paired with active anti-establishment-of-religion stance.
- **Class / rural** is not a CD signal by itself.

---

### CU — Cultural Uniformity
**Pole 1:** uniformity (one religion / language / worldview / conception of the good life is shared and enforced)
**Pole 5:** pluralism (different ways of life can coexist as equals; no single conception of the good)

**Broader scope** than narrow culture-only — includes religious, civilizational, lifestyle, family-structure, and civic pluralism.

| Score | Anchor examples |
|-------|-----------------|
| 1 | Klan, max-nativist platforms, ethno-nationalists |
| 2 | Assimilationist civic nationalism ("one America" creedal), most expansionist-era candidates, Wilson's internationalism (universalist civic creed, not pluralism) |
| 3 | Mixed / civic mainstream |
| 4 | Multicultural-positive D mainstream, civic-pluralist liberals |
| 5 | Radical multiculturalism, deep cosmopolitan pluralism (rare among nominees) |

**Common miscoding traps:**
- **Pattern B trap:** rhetorical universalism ("all men are created equal") ≠ CU=5 if the practiced scope is one creed/people. → Code by *practiced* breadth.
- **"Melting pot" is CU 2-3, not 5.** Melting-pot is assimilationist universalism (one shared identity); pluralism (5) accepts persistent difference.
- **Internationalism in foreign policy** ≠ CU pluralism. Wilson-style export-our-creed internationalism is CU 2.
- **American exceptionalism / "shining city"** rhetoric is usually CU 2 (one civic creed for all).

---

### MOR — Moral Circle
**Pole 1:** narrow moral circle (in-group only — concern bounded to family/tribe/nation/race)
**Pole 5:** wide moral circle (universalist — concern extends to all humans, possibly beyond)

**SPATIAL SCOPE ONLY.** This is *who counts morally*, not *what is morally good*. Progressive-vs-traditional moral content lives in CD; do not confuse.

| Score | Anchor examples |
|-------|-----------------|
| 1 | Klan candidates, slaveholders practiced (e.g., Calhoun-style explicit defense), ethnic-cleansing platforms |
| 2 | In-group-favoring policies that tolerate harm to out-group (Jefferson 1800 — slaveholder, Indian-Removal-sympathizer; Cass 1848 — popular-sovereignty slavery accommodation) |
| 3 | Civic-national: concern bounded to fellow citizens, no active harm to out-group |
| 4 | Liberal internationalist: concern for some out-groups (Marshall Plan, Truman, Carter human rights) |
| 5 | Max-universalist abolitionists who *acted* (Garrison, Sumner, Douglass), peace-corps idealists, effective-altruist worldview |

**Common miscoding traps:**
- **Pattern B:** rhetorical universalism ≠ practiced scope. Jefferson said "all men are created equal" while holding 600+ enslaved people. → MOR low, not high.
- **Religious moral language** is content (CD), not scope. "Christian morality" can pair with any MOR value depending on practiced scope.
- **Pacifism** + universal-rights *practice* = MOR high. Pacifism alone (without out-group concern) is not enough.
- **Slaveholding, ethnic cleansing, willingness to harm out-group for in-group benefit** = MOR low regardless of rhetoric.
- **Anti-immigration restrictionism** is MOR low (4 or below); pro-refugee internationalism is MOR high.

## MEANS Cluster

### PRO — Proceduralism
**Pole 1:** outcome-focused / anti-procedural (will break rules to get the result)
**Pole 5:** rules-bound / proceduralist (rules are the legitimate substrate)

| Score | Anchor examples |
|-------|-----------------|
| 1 | Trump 2020 (post-election actions), revolutionary anti-system actors |
| 2 | Outcome-focused but mostly within rules; Jackson's anti-Bank veto rhetoric |
| 3 | Mixed — most modern nominees |
| 4 | Bryan, La Follette, H. Wallace 1948, FDR (institutional reformers — *use* institutions hard) |
| 5 | Strict constitutionalists, court-revering proceduralists, Adams (J.Q.) |

**TR 1912 special case: PRO 3.** Recall of judicial decisions, executive activism, party rupture, plebiscitary style → mixed-procedural. Not anti-procedural, not clean institutionalist. Code TR 1912 as PRO 3, not 4.

**Common miscoding traps:**
- **Pattern A:** anti-establishment ≠ anti-institutional. Jacksonians, Bryans, La Follettes, TRs, H. Wallaces are *reformers within institutions* — they want to use the legal/political machinery harder, not abolish it. → PRO mid-high (3-4), not 1.
- **"Direct action" rhetoric** without rule-breaking *action* = PRO mid (3). Code on actions, not vibes.
- **Strict-constitutionalist** rhetoric = PRO 5 even when results are conservative-coded.
- **Authoritarian rule-by-decree** when it occurs = PRO 1 regardless of party.

---

### COM — Compromise Tolerance
**Pole 1:** uncompromising / principled (will not deal away principle for partial wins)
**Pole 5:** compromising / consensus-seeking (half a loaf is bread)

| Score | Anchor examples |
|-------|-----------------|
| 1 | Garrison (abolitionist purist), Norquist tax-pledge enforcement, single-issue movement candidates |
| 2 | Bryan (won't compromise on silver), Goldwater 1964 ("extremism in defense of liberty") |
| 3 | Most modern nominees |
| 4 | Cass 1848 (popular sovereignty *was* the compromise position), most Senate dealmakers |
| 5 | Henry Clay, Bob Dole, classic dealmakers, fusion-coalition candidates |

**Common miscoding traps:**
- **"Bipartisan" rhetoric** ≠ COM 5 if actions are partisan. Code on actions.
- **Single-issue purists** are COM 1 *in their domain*; may be COM mid overall if pragmatic elsewhere.
- **PRO-high ≠ COM-high.** A rule-bound proceduralist can also be a principled non-compromiser.

---

### EPS — Epistemic Style (categorical, 6 values)
| Index | Style | Signal |
|-------|-------|--------|
| 0 | empiricist | data, evidence, "let's measure it" — Obama, Carter, technocrat reformers |
| 1 | institutionalist | defer to professional/expert consensus — most party-establishment nominees, Cass, Adams (J.Q.) |
| 2 | traditionalist | defer to tradition / religion / elders — Bryan (biblical "cross of gold," providential democracy, agrarian inherited virtue), Reagan tradition appeals, religious-right candidates |
| 3 | intuitionist | gut / common sense — Jackson (military intuition), Trump 2016 |
| 4 | autonomous | think-for-yourself / contrarian — Jefferson, Ron Paul, libertarian outsiders |
| 5 | nihilist | reject the question / truth-is-power — late-stage Trump 2020, accelerationist outsiders |

**Common miscoding traps:**
- **Don't conflate ONT_S with EPS.** ONT_S is the belief that *political* institutions can act effectively; EPS is mode of *knowledge*-authority. They can move independently (Bryan: ONT_S high, EPS traditionalist).
- **Bryan is traditionalist, not institutionalist.** His authority claim was scripture and farmer common-sense, not professional consensus.
- **Trump shifts**: 2016 intuitionist; 2020 nihilist (depending on what you're coding).

---

### AES — Aesthetic Style (categorical, 6 values)
| Index | Style | Signal |
|-------|-------|--------|
| 0 | statesman | gravitas, formality, dignity-of-office — Washington, Lincoln, Eisenhower, Adams |
| 1 | technocrat | charts, expertise, neutral-competent — Carter, Dukakis, Romney, Mondale |
| 2 | pastoral | rural, plain, salt-of-earth — Carter (pastoral side), country-style appeals, Bryan 1908 |
| 3 | plainspoken | direct, unpolished, "talks like us" (was "authentic") — Truman, Biden, McCain |
| 4 | fighter | combative, scrappy, against-the-system — Jackson, Trump, Bernie at times |
| 5 | visionary | utopian, prophetic, future-oriented — Obama 2008, Reagan ("morning in America"), Wilson, Bryan 1896 ("Cross of Gold"), Bryan 1900 |

**Bryan AES is year-specific:** 1896 and 1900 are visionary (5); 1908 has shifted to pastoral (2). Don't mechanically code all three Bryan rows the same.

**Common miscoding traps:**
- AES is *style only*, not substance. Bernie's substance is left-economic (MAT 1) but his style oscillates between fighter (4) and visionary (5).
- Same candidate can carry different AES across years.

## REALITY Cluster

### ZS — Zero-Sum Orientation
**Pole 1:** positive-sum / cooperative (growth-mindset, win-win possible)
**Pole 5:** zero-sum / competitive (fixed pie, win-lose, "they win we lose")

| Score | Anchor examples |
|-------|-----------------|
| 1 | Reagan "morning in America," free-trade liberalism, Clinton 1992 New Economy |
| 2 | Most modern liberal-mainstream nominees, civil-rights inclusion-framers (Truman, LBJ Great Society, Humphrey) |
| 3 | Mixed |
| 4 | Trade protectionist + class-conflict mid-frame |
| 5 | Trump 2016 ("we win they lose"), max-zero-sum populist nationalism, civil-rights backlash framers (Wallace 1968, Thurmond 1948) |

**Common miscoding traps:**
- **Class-war rhetoric** without growth-vision = ZS high.
- **Free-trade liberalism** = ZS low.
- **Civil-rights expansionism (conditional rule):** ZS low (usually 2) when framed as inclusion / constitutional completion / "we all rise together" / Great Society / equal citizenship — Truman, LBJ, Humphrey-style. ZS high (4-5) when framed as racial threat / group displacement / "their gain is our loss" — Wallace 1968, Thurmond 1948 backlash style.
- **Tariff/protectionism** = ZS mid-high.

---

### ONT_H — Human Malleability via Cultivation
**Pole 1:** humans fixed; cultivation can't reshape them ("human nature is what it is")
**Pole 5:** humans malleable via education / institutions / tradition (perfectibility through cultivation)

| Score | Anchor examples |
|-------|-----------------|
| 1 | Scientific-racism essentialists, hard-libertarian "people are people," evolutionary determinists |
| 2 | Skeptical-of-reform conservatives |
| 3 | Mixed |
| 4 | Progressive reformers who believe in education |
| 5 | Confucian/Catholic moral-formation traditions, classical-liberal cultivation, Rousseau-Skinner behavioral-modification utopians |

**Common miscoding traps:**
- **NOT optimism vs. pessimism about politics.** That's a different axis.
- **NOT "good people vs. bad people."** It's a metaphysical claim about whether cultivation works.
- **Religious traditionalists who believe in moral formation** = ONT_H high (5), not low. The rubric measures malleability-via-cultivation, and traditional religion is one of the strongest cultivation regimes.
- **Libertarians/objectivists who believe people are who they are** = ONT_H low.
- **Scientific-racialists** = ONT_H low (1) because race is fixed in their worldview.

---

### ONT_S — Institutional Capacity Belief
**Pole 1:** institutional nihilism (libertarian, accelerationist, "system is rigged forever and can't be fixed")
**Pole 5:** institutional capacity belief (institutions can be made to work)

| Score | Anchor examples |
|-------|-----------------|
| 1 | Ron Paul, Murray Rothbard, accelerationists, anti-political "burn it down" |
| 2 | Strong critics of current institutions who haven't built alternatives — Jefferson 1800 (anti-Federalist) |
| 3 | Mixed |
| 4 | Bryan, La Follette, H. Wallace 1948 (institutional reformers who *use* institutions hard) |
| 5 | Hamilton, FDR New Dealers, Progressive-era state-builders, TR 1912 (New Nationalism — national institutions can solve collective problems), Adams (J.Q.) developmentalist |

**Common miscoding traps:**
- **Pattern A:** anti-establishment reformers are NOT institutional nihilists. They're often ONT_S high (4-5) because they want to *use* institutions to fix problems. Bryan ≠ Ron Paul.
- **Critique of CURRENT institutions** ≠ belief institutions can never work.
- **Accelerationist / nihilist rhetoric** → ONT_S low (1).
- **Don't read libertarian-policy preference as ONT_S 1.** Some Jeffersonian agrarians wanted weaker federal government but believed local institutions worked fine — that's ONT_S 2-3, not 1.
- **Don't mechanically apply the Jeffersonian-agrarian template to Madison/Monroe once they are governing institution-builders.** Jeffersonian factional origin is one thing; later presidential signal can drift toward higher ONT_S as they actually use federal institutions (war powers, internal improvements, doctrine-formulation).

## SELF Cluster (activation-only, per ADR-005)

### PF — Political Fusion
**Pole 1:** independent / non-partisan
**Pole 5:** strong partisan identity (party-is-identity)

`pos` doubles as activation; no separate `sal`.

| Score | Anchor examples |
|-------|-----------------|
| 1 | Eisenhower-style independents, third-way pragmatists |
| 2-3 | Most candidates |
| 4 | Strong-partisan mainstream nominees |
| 5 | Lifelong straight-ticket party-builders (Jefferson D-R, Jackson Democrat, modern strong-party nominees) |

**Trap:** PF is fusion of *identity* with party, not just party preference. A pragmatic Democrat with no emotional party attachment is PF 2-3, not 5.

---

### TRB — Tribe
**Pole 1:** universalist / low tribal attachment
**Pole 5:** tribal / strong in-group identity

The 9 anchor categories (national, ideological, religious, class, ethnic_racial, gender, sexual, global, mixed_none) describe the *kind* of tribe; TRB position describes *how strongly tribal* the candidate is.

| Score | Anchor examples |
|-------|-----------------|
| 1 | Cosmopolitan universalists, post-tribal pragmatists (rare among nominees) |
| 3 | Mixed civic-national identity without strong in-group framing |
| 5 | Strong in-group politics (Jackson "common man" white-male tribal, modern strong-tribal candidates) |

---

### ENG — Engagement Level
**Pole 1:** apolitical / disengaged
**Pole 5:** highly politically engaged / "politics is life"

For candidates, ENG is almost always 4-5 (you don't run for president while disengaged). The variance lives more on the respondent side. Reluctant-candidate signals (Taylor 1848, draft-Eisenhower) push ENG to 3.

---

## Period-specificity note

A 1796 candidate's CD score is *not* directly comparable to a 2020 candidate's CD score in absolute terms — the contested social-direction issues changed. Code each candidate against *their own period's contested questions*. Period explosion (the dataset already does this) keeps eras semantically separate when computing alignment, so within-period coding is what matters.

The same applies to CU: the breadth of "ways of life" considered politically legible changed dramatically across eras. Code 1796 candidates against the 1796-legible space (Federalist creedal vs. Jeffersonian states-rights diversity), not against modern multiculturalism.

## Workflow for the audit (Phase 3)

For each row in `candidates.ts` and `elections-*.ts`:

1. **Read the row.** Note current vector + comments.
2. **For each node**, run the independence test: would a careful historian recognize this value?
3. **Flag Pattern A** if PRO ≤ 2 + ONT_S ≤ 2 paired with anti-establishment-reformer signals.
4. **Flag Pattern B** if MOR ≥ 4 or CU ≥ 4 paired with practiced narrow-scope signals (slaveholding, ethnic restriction, in-group-only policy).
5. **Confidence rating:**
   - **HIGH** = ≥3 nodes wrong with clear historical citation; or single node wrong by ≥2 points with strong evidence.
   - **MEDIUM** = 1-2 nodes off by 1 point, debatable evidence.
   - **LOW** = vibes-disagreement only; defer.
6. **Output** to `results/candidate-audit/audit.csv` with columns: file, year, name, current_vector, proposed_vector, deltas, confidence, evidence_note.
7. **Apply only HIGH** in Phase 4. MEDIUM → second-pass review.
